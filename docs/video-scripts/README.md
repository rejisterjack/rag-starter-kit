## Production Guidelines

### Recording Setup

**Screen Recording**
- Resolution: 1920x1080 minimum
- Frame rate: 30fps
- Clean browser profile (no personal bookmarks)
- Large font size (14px+) for terminal/code

**Audio**
- Quiet recording environment
- Pop filter recommended
- Consistent microphone distance

**Code Display**
- Font: JetBrains Mono, Fira Code, or similar
- Syntax highlighting: Dark theme
- Line numbers enabled

### Editing Checklist

- [ ] Introduction hook (first 10 seconds)
- [ ] Clear audio, no background noise
- [ ] Smooth transitions between scenes
- [ ] Zoom on important UI elements
- [ ] Captions for accessibility
- [ ] Background music (low volume)
- [ ] End screen with CTAs

### Thumbnail Guidelines

- Eye-catching colors
- Clear, readable text
- Product screenshot or icon
- "5 min", "FREE", or other value indicators

## Publishing Platforms

- **YouTube**: Main channel
- **Vimeo**: Higher quality alternative
- **TikTok/Shorts**: Cut shorter versions
- **Documentation**: Embed in docs

## Script Format

Each script includes:
- **Overview**: Duration, audience, goal
- **Scenes**: Timestamps with visuals and narration
- **Production Notes**: Technical requirements
- **B-Roll**: Additional footage needed

## Contributing

To suggest a new video topic:
1. Create a new markdown file in this directory
2. Use the existing scripts as templates
3. Include all sections from the format
4. Submit a PR for review
