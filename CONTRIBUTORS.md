## How to Contribute

We welcome all types of contributions! See our [Contributing Guide](./CONTRIBUTING.md) for details.

### Ways to Contribute

1. **Code** - Submit PRs for bug fixes or features
2. **Documentation** - Improve docs, add examples
3. **Design** - UI/UX improvements, logos, graphics
4. **Testing** - Write tests, report bugs
5. **Ideas** - Suggest features, provide feedback
6. **Community** - Help others in discussions

## Adding Contributors

This project follows the [All Contributors](https://allcontributors.org/) specification. To add a contributor, comment on an issue or PR:

```
@all-contributors please add @username for code, doc
```

Or use the [All Contributors CLI](https://allcontributors.org/docs/en/cli/installation):

```bash
npx all-contributors add @username code,doc
```

Then regenerate the table:

```bash
npx all-contributors generate
```
