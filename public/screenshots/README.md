## 🎨 Screenshot Guidelines

1. **Resolution**: 1920x1080 or 1440x900
2. **Format**: PNG for screenshots, GIF for animations
3. **Compression**: Use TinyPNG for optimization
4. **Annotations**: Add arrows/text if needed (use Figma/Sketch)
5. **Consistency**: Same browser, same zoom level (100%)

## 🛠️ Tools

- **Screenshots**: CleanShot X (Mac), ShareX (Windows), Flameshot (Linux)
- **GIFs**: ScreenFlow, Loom, or Giphy Capture
- **Editing**: Figma, Canva, or Photoshop
- **Optimization**: ImageOptim, Squoosh
